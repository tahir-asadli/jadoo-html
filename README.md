# Jadoo - Travel Landing Page

A modern travel landing page built with HTML, Tailwind CSS v4, Alpine.js, Splide.js, and AOS animations.

**[Preview](https://jadoo-html.netlify.app/)**

## Tech Stack

- **HTML5**
- **Tailwind CSS v4**
- **Alpine.js** — reactive UI components (mobile menu, language dropdown)
- **Splide.js** — partners carousel
- **AOS** — scroll animations

## Getting Started

Install dependencies:

```bash
npm install
```

Build CSS:

```bash
make tailwind-build
```

Watch for changes:

```bash
make tailwind-watch
```

Then open `index.html` in your browser.
